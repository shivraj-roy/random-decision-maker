# Random Decision Maker

Make decisions with dramatic flair! This Raycast extension helps you make choices when you're stuck – whether it's a simple Yes/No question or picking from multiple options.

## Features

### Decide Yes or No
Ask any yes/no question and let the universe decide!

- Type your question and submit
- Watch the 3-second suspenseful thinking animation with loading indicator
- Get your answer with a clear checkmark (green for YES) or X (red for NO)
- Fun randomized response messages like "The universe approves!" or "Not today, friend."

### Spin the Wheel
Can't decide between multiple options? Add them all and spin the wheel!

- Add as many options as you need (minimum 2)
- Watch the 3-second animated spin through your choices
- Get a random winner with trophy icon and celebration message
- Save presets for frequently used option sets
- Delete individual options with `⌘ + Backspace` (option 3 and above)
- Clear all options with `⌘ + Shift + Backspace`

### Decision History
Keep track of all your past decisions.

- View all previous Yes/No answers and wheel spins
- See when each decision was made (Today, Yesterday, X days ago)
- Color-coded icons: green checkmark for YES, red X for NO, yellow star for wheel results
- Delete individual entries with `⌘ + Return`
- Clear all history with `⌘ + Shift + Return`

## Commands

| Command | Description |
|---------|-------------|
| Decide Yes or No | Get a dramatic Yes/No answer to your question |
| Spin the Wheel | Pick randomly from multiple options |
| Decision History | View and manage past decisions |

## Keyboard Shortcuts

### Spin the Wheel
| Shortcut | Action |
|----------|--------|
| `⌘ + N` | Add more options |
| `⌘ + Backspace` | Delete focused option (option 3+) |
| `⌘ + Shift + Backspace` | Clear all options |

### Decision History
| Shortcut | Action |
|----------|--------|
| `⌘ + Return` | Delete selected entry |
| `⌘ + Shift + Return` | Clear all history |
