# Random Decision Maker

A Raycast extension that helps you make decisions with dramatic flair!

## Features

- **Decide Yes/No**: Ask questions and get randomized Yes/No answers
- **Spin the Wheel**: Add multiple options and spin to pick one randomly
- **Decision History**: View all your past decisions

## Installation

1. Open Raycast
2. Search for "Random Decision Maker"
3. Install the extension

## Usage

- Run `Decide Yes or No` to ask a question
- Run `Spin the Wheel` to choose from multiple options
- Run `Decision History` to see past decisions
